About Us


OpenSecure.org is a FREE service started on 01 Oct 2011 that allows users to scan a website address to determine whether it is making malicious connections to potentially damaging websites on the Internet or whether it contains malicious code.

Note: Even if a site shows up as benign from one scan, it is NOT guaranteed that the website is safe to visit.

If you want to contact us for more information or to provide feedback please send an email to the following address: info (at) opensecure (dot) org.


The OpenSecure.org website uses a open source plugin called Fireshark to retreive website information, more information can be found at fireshark.org
